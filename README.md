


Pig Game
A classic two-player dice game built with HTML, CSS, and JavaScript. This project demonstrates fundamental web development skills and provides an entertaining way to explore basic programming concepts.

Features
Interactive Gameplay: Enjoy a simple and engaging user interface for two players to take turns rolling dice and accumulating points.
Dynamic Scoreboard: Track each player's score and current round score in real-time.
Winning Conditions: Set a custom winning score or play to the default score of 100 points.
Responsive Design: Responsive layout ensures an optimal gaming experience on various devices.
Technologies Used
HTML5: Provides the structure for the game.
CSS3: Adds styling to create an attractive and user-friendly interface.
JavaScript: Powers the game logic, including dice rolling, score calculation, and player turns.
